## About

CSE student at Galgotias University (2026 batch), currently working as a **Technical Trainee at Bunk To Brains**, architecting scalable web platform features with Node.js, Express.js, and MongoDB.

Previously interned as a **Tech Intern at Innovexa Solutions**, building internal analytics dashboards with JavaScript, HTML, and CSS, and automating real-time data flow across teams via REST APIs.

I'm a fresher looking for an SDE / full-stack role at a product company or funded startup — but I'd rather let what's shipped below speak for that than a list of adjectives.

## What I'm building

| Project | What it is |
|---|---|
| **Career-Copilot** | A full-stack AI career platform (Next.js, TypeScript, Express.js, MongoDB) integrating OpenAI GPT-4o-mini for ATS scoring, resume analysis, and personalized career insights. Includes a JD-matching engine that computes resume-to-JD match %, detects skill gaps, and cuts manual review effort by ~70%. |
| **POOL-UP** | A full-stack carpooling web app (Node.js, Express.js, MongoDB) with dual dashboards, bcrypt authentication, and a CO₂ savings tracker across EV, CNG, and Petrol vehicles. Deployed live on Render with full CI/CD. |

Both are self-driven builds — no vanity metrics here, just working software with real auth, real databases, and real deployments I can walk through end to end.
